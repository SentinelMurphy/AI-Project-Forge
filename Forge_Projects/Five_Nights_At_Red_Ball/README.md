# Five Nights At Red Ball

A thrilling maze survival game where you must navigate through a labyrinth
while avoiding killer red balls. Every 20 seconds, a new red ball is added
to the maze, increasing the difficulty. Your goal is simple: find the exit
before the red balls catch you!

## Game Features

- **Survival Horror Maze**: Navigate through a dark maze with limited visibility
- **Increasing Difficulty**: A new red ball spawns every 20 seconds
- **Tense Gameplay**: Race against time to find the exit before being caught
- **Simple Controls**: Easy to learn, challenging to master

## Installation & Setup

Follow these steps to get the game running on your local machine:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/[your-username]/Five_Nights_At_Red_Ball.git
   ```

2. **Navigate to the Project Directory**
   ```bash
   cd Five_Nights_At_Red_Ball
   ```

3. **Install Dependencies and Build the Game**
   ```bash
   npm run build
   ```

4. **Start the Development Server**
   ```bash
   npm run dev
   ```

5. **Open Your Browser**
    - Once the server is running, open your web browser
    - Navigate to: `http://localhost:5173/`
    - Click "New Game" to begin your survival challenge

## How to Play

- Use **WASD** or **arrow keys** to navigate through the maze
- Avoid the red balls that patrol the corridors
- Every 20 seconds, a new red ball will be added to the maze
- Find the exit before the red balls catch you!
- The game ends when you either escape or get caught by a red ball
